# Pixenbuy

An e-commerce web application.

## Project Structure

```
pixenbuy/
├── client/    ← React frontend (Vite + TypeScript)
├── server/    ← Express backend (Node.js + Mongoose)  [coming soon]
```

## Getting Started

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
